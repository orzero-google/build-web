content：
0、自增id
1、标题
8、描述

3、作者id（邮箱、手机）
4、作者名
4、分类id （层、值）
5、区域id	（层、值）
6、关键词id	（值）
7、配置id		（类型、key、value（扩展））

	
9、内容(MD5散列目录.压缩文件)

10、状态id
11、权限id

12、上级id（内部ref）
13、url_ref
14、插入时间
	CREATE TABLE `contents` (
	`contentsid` int(11) NOT NULL auto_increment,
	`title` VARCHAR(255) NOT NULL,
	`desc` VARCHAR(255) NOT NULL,
	`nickname` VARCHAR(255) NOT NULL,
	`user_id` INT NOT NULL,
	`cate_id` INT NOT NULL,
	`area_id` INT NOT NULL,
	`keys_id` BIGINT NOT NULL,
	`conf_id` INT NOT NULL,
	`content` VARCHAR(255) NOT NULL,
	`mode_id` TINYINT NOT NULL,
	`perm_id` TINYINT NOT NULL,
	`ref_id` BIGINT NOT NULL,
	`url` VARCHAR(255) NOT NULL,
	`time` TIMESTAMP NOT NULL, PRIMARY KEY  (`contentsid`)) ENGINE=MyISAM;
	
	
状态:定义文档状态(status：1自己、2指定、3排除、0所有)
permissions：
0、自增id
1、status
2、type（0useid,1groupid(out),2groupid(slef)）
3、value (array)
	CREATE TABLE `permissions` (
	`permissionsid` int(11) NOT NULL auto_increment,
	`status` TINYINT NOT NULL,
	`type` TINYINT NOT NULL,
	`value` TEXT NOT NULL, PRIMARY KEY  (`permissionsid`)) ENGINE=MyISAM;


modes：
自增id
status 0,1,2,3,4,5
value 草稿,发布,回收站,关闭,删除
	CREATE TABLE `modes` (
	`modesid` int(11) NOT NULL auto_increment,
	`status` TINYINT NOT NULL,
	`value` VARCHAR(255) NOT NULL, PRIMARY KEY  (`modesid`)) ENGINE=MyISAM;


配置:
key定义文档属性(pdf,word,text...)
config：
0、自增id
1、type （0dir、1content）
2、key
3、value
	CREATE TABLE `configs` (
	`configsid` int(11) NOT NULL auto_increment,
	`type` INT NOT NULL,
	`key` VARCHAR(255) NOT NULL,
	`value` TEXT NOT NULL, PRIMARY KEY  (`configsid`)) ENGINE=MyISAM;
